Detailed Description

Disambiguator tag.

Empty struct tag type used to disambiguate between different functions taking a list of objects for when initializing a container. This is used as an alternative to std::initializer_list but without the overhead of std::initializer_list. For more information see the presentation by Jason Turner at C++Now 2018.
